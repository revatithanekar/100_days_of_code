
# 🚀 100 Days of Code – Full-Stack Developer Journey

## 🎯 Goal

To become a disciplined full-stack developer by building real projects using:

* HTML, CSS, JavaScript
* Node.js + Express.js
* MongoDB

---

## 📅 Rules

* Code **every day** (minimum 60 minutes)
* Make at least **1 meaningful commit daily**
* No zero days
* Build projects — not just watch tutorials
* Track daily progress

---

## 🧠 Daily Log Format

Day X:

* Learned:
* Built:
* Problem:
* Solution:
